# Molfi App Hashkey

![TypeScript](https://img.shields.io/badge/TypeScript-informational?style=flat-square)

> Molfi secondary web app on HashKey Chain

## Tech Stack

**ethers.js** · **Firebase** · **React** · **Tailwind CSS** · **viem** · **wagmi** · **TypeScript**

## Getting Started

```bash
# clone
git clone https://github.com/nickthelegend/molfi-app-hashkey.git
cd molfi-app-hashkey

npm install
npm run dev
npm run build
npm run test
```
